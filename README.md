## 測試的安裝步驟
### go install
https://golang.org/doc/install

### git add remote
```
git remote add origin git@github.com:a6976933/SE5-Backend-Websocket.git
```
### git pull
```
git pull origin master
```
### go path setting
```
export PATH=$PATH:yourGoInstallPath/go/bin
```
### download package
```
cd projectdir
```
```
go mod download
```
### build
```
go build main.go
```
### run
```
./main
```
